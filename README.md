# Taller-teste
Desenvolvimento do teste técnico da empresa Taller. Requisitos [aqui](https://github.com/TallerWebSolutions/teste-tecnico).<br>

## Execução

### API
```
cd api
npm install
node server
```

### SPA
```
cd spa-churras
npm install
npm start
```
